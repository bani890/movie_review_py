1.General information: This is a movie review website that uses PYTHON for the back end.
2.AUTHORS: Bnaibrata Das(bani890@gmail.com)
3.THANKS: UDACITY for the stater code.
Steps to deploy the project:
1. Run the entertainment_center.py. It contains open_movies_page() which opens the webpage in a browser. It takes list of movies as an input.
2. Two nodules have been imported: media.py and fresh_tomatoes.py.
3. media.py contains the classes and functions of the movie class.
4.fresh_tomatoes.py contains HTML templates,css and javascript code.
5.create_movie_tiles_content() can be used to display more properties of the movie class.
